# 📐 微积分从零开始 — Calculus From Zero

> 一套完整的微积分自学课程：从高中代数基础开始，一路走到多元微积分。

## 🧭 课程定位

本课程不是机械刷公式，而是建立一套**稳定的微积分判断流程**。每单元按"看对象 → 看限制 → 选工具 → 算结果 → 查意义"的流程推进。

完整主线：

```
函数 → 极限 → 导数 → 积分 → 级数 → 微分方程 → 多元微积分
```

## 📖 使用方式

### 推荐学习方法

1. **按顺序学习**：从第 0 单元开始，逐步推进。前面阶段（代数、函数、三角）是后面所有内容的基础。
2. **配合总目录**：先读 `微积分长期学习计划_v2.0_总目录提纲.md`，了解全貌后再选择当前学习的阶段。
3. **速查手册作伴**：学习过程中遇到遗忘的概念或公式，直接查阅 `微积分全课程快速查询手册_v1.0.md`，不必回翻旧单元。
4. **可配合 Obsidian**：所有文件均为 Obsidian 兼容的 Markdown 格式（含 Frontmatter），可直接导入 Obsidian 知识库。

### 前置知识

- 高中代数基础（数轴、区间、方程、不等式）
- 对函数有基本认识（定义域、值域、图像）
- 对三角和指对数有初步接触更佳（课程中也会系统覆盖）
